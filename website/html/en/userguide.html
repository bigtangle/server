User Guide

1
Please download the software from bigtangle.net. extract the zip file bigtangle.zip and run bigtangle.exe
If you have old version installed on your computer, you will be asked to upgrade.

2
On the main window，the balance sheet will displayed for all your keys in detail and summary。 Click the search to refresh the contents。For the selected records in balance, you can click the block info to display the content of the block.

3
In main window，click payment to start transfer.
Please enter the amount, token, send to and to start send.
4
Please select the Multi Sign to start payment which requires multi signatures payment to spent the output.
5
Please select multi address to pay token to multi address
Add Public Key to set the multi address
6
select the multi signatures and address
Save the data and then goto sign to start payment process.
7
Please select Sign tabs.
select the transaction, then click sign to process it. If the required number of signatures are processed. Then the transaction is finished.
8
Click search to find all orders with the status and account.
9
10
After order match you can find your matched orders to sign in the tabs Sign. The order is closed, if all parties have signed the matched order, You can cancel your signed order, if your partner does not sign the order in time.
11
Select Exchange
User can do any exchange in his manner，click the export after enter the required exchange information and save it as file. you can then send the file to your trading partner. The trading partner can then import this file and sign the transaction. Then save it in Bigtangle. You can click the refunds to revoke this transaction, if the transaction is pending.
12
Windows Token
click search to refresh the page，select a token and you can watched this token. The watched token list is encrypted as storage block in Bigtangle. To prevent spams, only the watched token and system token will displayed in balance sheet.
