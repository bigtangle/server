Quick Start for BigTangle Wallet

Use Cases

Payment
You must have something to start a payment. But you can pay any tokens to other address.
Tokens
Token can be anything from Fiat money to real products like computer. Anyone can issue any number of token. Let us first create your work days for track and trading. Click the create button in token window.
Enter the name of the token: My Work Days
Enter the amount of the token: 365
Select an address for this token as issuer.
Markets
Markets can be for stocks, bonds etc. can easily be realized by creating new token equivalents. Companies can publish stocks and use the BigTangle network. The existing stock markets can be fully integrated.
Supply Chain
Under the assumption of trustworthy suppliers issuing authenticity tokens, it is trivial to track product authenticity via token transfers. This allows the trustless tracking of inventories in supply chains for consumers and middlemen alike.
Decentralized E-commerce Platforms
All products and services can be tracked on the network, enabling anyone to build decentralized e-commerce platforms like Amazon or Alibaba.
