Quick Start for BigTangle Wallet

Use Cases

Payment
You must have something to start a payment. You can pay any tokens to other address as same as system coin BIG. You can select a token from the list and enter an amount and address to pay. Then click the button Send. After the payment, the system will ask you to save the address in the contact list. You can select the address next time from this list.
Tokens
Token can be anything from Fiat money to real products like computer. Anyone can issue any number of token. Let us first create your work days for track and trading. Click the create button in token window.
Enter the name of the token: My Work Days
Enter the amount of the token: 365
Select an address for this token as issuer.
Then save it.
Markets
Markets can be real stock exchange for stocks, bonds etc. Financial products can easily be realized by creating new token equivalents. The system has a integrated distributed free market for trading service. In last step we create a work day token. Now we want get system coin for the hard work. Your can click the order tab in the window market. Then select your token and give a price limit for your work days. Now you can wait until someone to pay the token. In the test we have an agent running as automate service. The agent will buy any tokens for sell.
Exchange
If the order is matched, by default it is only a deal to be signed in the Bigtangle.net In the list you will find the deal and it is already signed by the automate agent. You click the sign button and the deal is finished now and you will get the system coins. Your can do exchange any token. Click create in the exchange window, you can do make a deal to anyone in any form.
Blocks
By default it will display all blocks related to your payment. If you do the transfer of token, but it does not show in the balance main window, then you will find the blocks here with rating lower then 75%, These blocks may be in a conflicts or there is delay in the server processing. You can select the block show the details of the block.
Multi Sign
It is possible to set payment as required N of M sign. For examples, it may require two signatures to authorize a transaction. You do first sign by click export, then send it to next one. The next user import the block file and click the send to finish the payment.
Setting
Bigtangle wallet is a key store. It saved all your public and private keys in your wallet. You can set a password to protect your wallet, but IT IS IMPOSSIBLE DO RECOVERY THE WALLET WITHOUT YOUR PASSWORD. Your wallte is per default saved in your home directory with your usename.wallet for example C:\Users\cui\cui.wallet. The file of your contact list and your favorite token list are same on the same directory.
