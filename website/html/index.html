BigTangle BigTangle is a cryptocurrency network extending the directed acyclic graph architecture. Through the use of industry standard big data technology in conjunction with the parallelizable architecture, it is intended for nodes to run on computer clusters, therefore no longer being limited by single machine performance. The transaction consensus is achieved in a completely decentralized manner and custom token self-issuances are supported to enable a number of economic key use cases via sufficient throughput.

Use Cases

Payment
By providing a scalable infrastructure, BigTangle enables the global transaction volume to be processed in one network. Most importantly, this offers infrastructural cost ad- vantages by eliminating complex and costly processes of traditional payment processing for banks, companies and the general populace.
Fiat Money (conventional Currency)
For banks and central banks, the token issuance protocol can be used to issue bank-backed tokens denoting conventional fiat money. Fiat money transactions can then feasibly be processed within seconds on a global scale.
Stock Markets
Markets for stocks, bonds etc. can easily be realized by creating new token equivalents. Companies can publish stocks and use the BigTangle network. There is integrated free market to trade service. The existing markets like NY Exchange, can be fully integrated.
Free Markets
There is a integrated markets for any token issued on the BigTangle and it is fully free to match any token trade.
Supply Chain
Under the assumption of trustworthy suppliers issuing authenticity tokens, it is trivial to track product authenticity via token transfers. This use case extends into classic supply chain management as well, allowing the trustless tracking of inventories in supply chains.
Decentralized E-commercial Platform
As all products and service are tracked by supply chain management. Anyone can build decentralized e-commercial platform like Amazon and Alibaba.

Frequently Asked Questions

What is BigTangle?

BigTangle is a novel big data approach to scalable cryptocurrencies employing state-of-the-art big data technologies in combination with the highly scalable Tangle directed acyclic graph base architecture is proposed, thereby achieving high performance in terms of throughput by parallelized computations on a general directed acyclic block graph instead of a blockchain.

What are the features of BigTangle?

Baseline properties provided by BigTangle are high hash power security due to the mining process, infinite scalability, sufficiently fast transaction confirmation times, full decentralization, trustlessness and permissionlessness, feeless transactions, in-principle quantum security and normalized inflation.

Why is it called BigTangle?

Our solution adapts the principle of tangle in a more powerful environment and increases the performance by using state-of-the-art big data technologies.

What are potential use-cases?

Projected practical use cases include payment processing, decentralized stock exchanges, supply chain and product integrity tracking.
